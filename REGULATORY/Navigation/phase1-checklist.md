# Phase-1 Regulatory Checklist

## IRB / Vet Approval
- ☐ Application submitted  
- ☐ Consent attached  
- ☐ Device description included  
- ☐ Safety interlocks described  
- ☐ Investigators listed  

## Device Requirements
- ☐ Calibration logs  
- ☐ SAR module enabled  
- ☐ Fail-safe tested  
- ☐ Versioning documented  

## Data & Privacy
- ☐ PHI removed before storage  
- ☐ Study IDs assigned  
- ☐ Access limited to approved staff  

## Reporting
- ☐ Adverse events logged  
- ☐ DSMB notified
